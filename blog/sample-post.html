Welcome to the blog

This is a sample post. To add a new post:

  1. Create a new HTML file in the blog/ folder (e.g. my-post.html).
  2. Copy the structure of this page: same header and footer, section#blog-post with .post-meta, h2, and your content.
  3. Add an entry to blog/index.html in the .blog-list list with the date, title, excerpt, and link to your new file.

Keeping the same header, nav, and footer ensures consistent formatting across the site.
